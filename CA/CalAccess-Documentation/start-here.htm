California Secretary of State Political Reform Division

The California Secretary of State releases this raw campaign finance and lobbying activity data submitted in accordance with Political Reform Act filing requirements. The data is available in ASCII tab-delimited text format. This allows experienced users to import the data into databases and analysis tools.

To aid people in using the data, the Secretary of State provides descriptions of the CAL-ACCESS database tables; however, users should also obtain copies of the original Fair Political Practices Committee (FPPC) forms. The California Electronic Filing Format documents used in the data entry are included. The Secretary of State provides descriptions of the mapping between these documents and the database tables. To read these documents you will need an Adobe Acrobat (PDF) Reader.

Please send feedback and questions to the Secretary of State’s Political Reform Division (PRD).
